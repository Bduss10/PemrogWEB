File HTML ini digunakan untuk membuat halaman profil pribadi sederhana menggunakan HTML dasar.

Struktur dimulai dengan <!doctype html> yang menandakan bahwa dokumen menggunakan standar HTML5. Pada bagian <head>, terdapat pengaturan karakter (UTF-8), viewport agar responsif di berbagai perangkat, serta judul halaman My Profile yang tampil di tab browser.
Di dalam <body>, halaman diawali dengan judul Welcome to My Profile dan deskripsi singkat tentang halaman profil. Selanjutnya terdapat menu navigasi menggunakan tag <a> yang mengarah ke beberapa bagian dalam halaman seperti Tentang Saya, Pendidikan, Keahlian, dan Kontak dengan bantuan atribut id.
Bagian Tentang Saya menampilkan foto profil menggunakan <img> dan paragraf perkenalan singkat yang memanfaatkan tag <b> dan <i> untuk menebalkan dan memiringkan teks.
Pada bagian Riwayat Pendidikan, digunakan tabel <table> untuk menampilkan data tahun, institusi, dan jurusan secara terstruktur. Tabel ini terdiri dari baris header dan beberapa baris isi.
Bagian Keahlian berisi daftar kemampuan yang ditampilkan menggunakan list tidak berurutan (<ul>) untuk bahasa pemrograman dan list berurutan (<ol>) untuk hobi atau aktivitas.
Terakhir, bagian Kontak menggunakan <form> yang berisi input nama, email, dan pesan. Form ini menggunakan atribut type, id, dan name untuk mendukung pengolahan data form. Halaman ditutup dengan footer sederhana berisi copyright dan tautan ke GitHub serta Instagram.
